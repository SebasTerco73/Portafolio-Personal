Sobre mí

Soy estudiante de desarrollo de software, motivado por entender cómo funcionan las cosas más allá de la superficie. Me interesa crear soluciones web que no solo funcionen, sino que sean claras, eficientes y pensadas para las personas que las usan. Disfruto aprender constantemente, enfrentar desafíos técnicos y mejorar con cada proyecto. Mi misión como desarrollador es construir herramientas útiles, seguir creciendo profesionalmente y aportar valor real a través del código.

Mis Proyectos

Elektro-ventas es una página web orientada a presentar productos de forma clara y atractiva. Está desarrollada con React, lo que permite una interfaz moderna, dinámica y reutilizable por componentes. Incluye una estructura pensada para que el usuario encuentre información rápidamente.

Es una aplicación de escritorio para la gestión de un restaurante, con inicio de sesión segun rango y pantallas para administrar la información del sistema. La implementé en Java 8 usando Swing para la interfaz y una arquitectura organizada en entidades y DAOs para el acceso a datos. Persistí la información en MySQL/MariaDB.

Es un videojuego 2D con temática de piratas, hecho con python y pygame, donde el jugador controla a un pirata que debe navegar por diferentes niveles enfrentando enemigos y obstáculos. Implementé mecánicas como movimiento lateral, barra de vida, música y efectos, pantalla de inicio con instrucciones y sistema de puntaje.
