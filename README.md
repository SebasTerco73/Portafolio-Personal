Sitio web de portafolio personal desarrollado con HTML5, CSS3 y JavaScript vanilla.

[Ver en vivo](https://portafolio-personal-sebasputche73.vercel.app) <!-- actualizar con la URL real si es diferente -->

## 📋 Secciones

- **Sobre mí** — Breve presentación personal y motivaciones como desarrollador.
- **Skills** — Tecnologías y herramientas que manejo (HTML5, CSS3, Bootstrap, JavaScript, Node.js, React, Java, C# / .NET, Python, Android, Git, GitHub, MySQL, Postman, Supabase, Figma, Vercel, VS Code).
- **Hobbies** — Intereses personales: fútbol, videojuegos, música, guitarra, lectura y programación.
- **Películas** — Películas favoritas con descripción.
- **Discos** — Álbumes favoritos con enlace a YouTube.
- **Contacto** — Formulario de contacto.

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura del sitio |
| CSS3 | Estilos y diseño responsivo |
| JavaScript | Interactividad (menú hamburguesa, botones de scroll) |
